iKindle - Manage Kindle Highlights & Notes

Posted on 2017-02-11   |  

iKindle is a new way to manage Kindle highlights and notes, help you to gain more from reading.

1. Start From Import

Just connect your Kindle to Mac, select Kindle folder, and start Import Kindle highlights & notes.

You only need to manually import for one time. Later, iKindle will automatically import when you connect Kindle.

2. iKindle: Live for Notes

Focus on Now

The books we ever read become more and more. If put all these books together, we will be lost.

iKindle put read books in a group, which can be collapsed. It’s a small feature, but could help you to focus on current reading.

Review Notes

Just press Space key, you can quickly review the note. Feel familiar?

iKindle could review notes one by one, help you to really understand them and gain more from reading.

Free to Edit

You will feel free to edit book title, author, highlights, notes, etc.

Search Anything

You can search any possible content:

  • Book title
  • Author
  • Note
    • Content
    • Location
    • Date

Take an example, if I want to search the note with ‘control’ in book ‘Steve Jobs’, just input ‘Jobs control’.

Copy as Markdown

It will be more valuable to share notes with others. Just copy notes as Markdown, and then you can easily share them in your blog or anywhere else.

In fact, this is why I developed iKindle. After reading a book, I would like to summarize it in my blog. And the highlights and notes are important parts of the summary. So, I made iKindle to copy them as Markdown.

3. Great Experience

It’s not enough just work, should work better.

Native Interaction

iKindle works like the built-in app Reminder, which is very suitable for notes management.

So, you’re already an iKindle expert.

Fullscreen is Cool

It’s hard to describe the feeling, but you will feel cool to review notes with iKindle in fullscreen.

Press ‘Command + Ctrl + F’ to have a try.

Efficiency with Shortcuts

iKindle support a variety of shortcuts.

  • Tab Switch among books, notes and search.
  • Space Quick look
  • Arrow Switch nearby book or note
  • J K Works as arrow key. Vimer will like it.
  • Command + F Search
  • Command + I Get Info
  • Command + C Copy
  • Command + Shift + C Copy as Markdown
  • Command + Shift + R Mark as read
  • Command + Option + S Hide sidebar
  • Command + Ctrl + F Enter fullscreen
  • Command + Shift + I Import
  • Command + Shift + E Export
  • Ctrl + F Feedback
  • Ctrl + Option + F Feedback by email
